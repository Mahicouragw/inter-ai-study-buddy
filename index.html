Inter AI Study Buddy

Telangana Intermediate 1st & 2nd year — Commerce, Economics, Civics, Accountancy, English, Telugu

📂 Source on GitHub 📖 Install Guide (README)

✨ What it does

  • 🤖 AI Tutor chat (free Gemini key) with model fallback — asks answered for Inter syllabus
  • 🧒 "Explain like I'm five" mode for every answer
  • ✍️ Exam Answer Writer — 2-mark, 5-mark & 10-mark answers, with offline fallback
  • 📚 12 subjects (6 subjects × both years), chapters, key points, Q&A and MCQs
  • 🃏 110-word vocabulary builder with Telugu meanings, flashcards & quiz
  • 🎙️ Fully voice-friendly: speak questions (speech-to-text) and listen to answers (text-to-speech) — great with TalkBack
  • 📄 Direct links to official free TOSS/TSBIE PDFs & model papers

📲 Get the app (Android)

  1. Clone the repo: git clone https://github.com/Mahicouragw/inter-ai-study-buddy.git
  2. Run flutter create --platforms=android . then flutter pub get
  3. Build: flutter build apk --debug and install on your phone
  4. Add a free Gemini API key in Settings (link inside the app)

Everything is documented step-by-step in the README, including TalkBack tips.
